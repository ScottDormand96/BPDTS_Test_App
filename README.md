# BPDTS_Test_App
A test application to find solution to: Using the language of your choice please build your own API which calls the API at https://bpdts-test-app.herokuapp.com/, and returns people who are listed as either living in London, or whose current coordinates are within 50 miles of London.

# Prerequisites
- Visual Studio
- .Net Framework

# Dependencies
- NUnit
- NUnit3TestAdapter
- Moq
- jquery
- jquery.validation
- bootstrap
- Mvc
- Razor
- AspNetWebStack
- Newtonsoft.Json

# Run App
- Clone from: https://github.com/ScottDormand96/BPDTS_Test_App
- Click BPDTSTestApp.sln within application folder in VS
- Start Debugging (F5)
- The application will then run in selected browser at the follwing URL: https://localhost:XXXXX/
